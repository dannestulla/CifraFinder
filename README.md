# CifraFinder
Finds the tablature of the song being played on Spotify, a great program for playing guitar while following along with the music. The app was fully integrated with Bitrise, and each pull request and merge into the main branch generates a build. Deployment to the store via Bitrise with Fastlane has been implemented.

## Tools Used:
Spotify API and Google Custom Search API
Work Manager and Jetpack Compose
Clean architecture, MVVM, Koin
Integration with Bitrise as CI/CD with Fastlane for store deployment
Unit tests
## Download Apk Link
https://play.google.com/store/apps/details?id=br.gohan.cifrafinder
